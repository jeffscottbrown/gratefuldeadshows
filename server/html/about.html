Grateful Dead Shows is a simple web application for browsing Grateful Dead concert information.

The site is free and open source. The source code is available in the GitHub repository at github.com/jeffscottbrown/gratefuldeadshows. Contributions are welcome.

The data provided in the application was sourced from github.com/jefsmit/gdshowsdb.
